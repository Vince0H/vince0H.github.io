      Java多线程学习

      Post on 10月 10 2014
      • 进程和线程
      • 多线程程序的运行模式
      • 定义线程的方法

      多进程：在操作系统中能同时运行多个任务（程序）

      多线程：在同一个应用程序中有多个顺序流（同时）执行

      创建线程的方法

      @Vindroid

      方式 1，创建两个类test.java和firstthread.java

      class test{
    public static void main(String[] args) {
        //生成线程类的对象;
        firstthread ft = new firstthread();
        //启动线程
        ft.start();
        for(int i=0;i<100;i++){
        System.out.println(i);
        }
    }
}    

      class firstthread extends Thread{
public void run(){
    for(int i=0;i<5;i++){
        System.out.println("线程测试");
    }
}
}

      方式二：同样创建两个类，使用了同步锁，synchronized():

      class MyThread implements Runnable{//该类实现了Runnable接口
    int i = 100;
    public void run(){//复写run方法
        while(true){
            synchronized(this){//可以确保获得this的线程可以顺利执行完下面的代码，确保安全性，为对象加锁
                System.out.println(Thread.currentThread().getName()+"-->"+i);
                i--;//
                Thread.yield();//让出CPU 
                if(i<0){
                    break;
                }
            }
        }
    }
}

      class test1{
    public static void main(String[] args) {
        MyThread myThread = new MyThread();
        //生成两个Thread对象，但是这两个Thread对象共用一个线程体
        Thread t1 = new Thread(myThread);//新建两个进程
        Thread t2 = new Thread(myThread);

        //每一个线程都有名字，可以通过Thread对象的setName（）方法设置线程名字，也可以使用getName方法获得线程的名字
        t1.setName("线程1");
        t2.setName("线程2");

        //分别启动每个线程
        t1.start();
        t2.start();
    }
}

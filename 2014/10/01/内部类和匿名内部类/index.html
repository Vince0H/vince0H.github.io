      内部类和匿名内部类

      Post on 10月 1 2014
      • 什么是内部类
      • 内部类的使用方法

      • 匿名内部类的使用方法

        首先生成创建一个内部类A.java：

        class A{

          int i;
  class B{
      int j;
      //B可以使用A的成员变量，但是B没有拥有A的成员变量
      int funB(){
          int result  = A.this.i+this.j;
          //两个this有区别
          return result;
      }
  }

        }

      测试用的主函数Testing.java

      class Testing{
    public static void main(String args[]){
        A a = new A ();//创建A对象

    A.B b = a.new B();//创建内部类的B对象
    }
}

      匿名内部类:

      interface A{
    public void do();//在匿名类中定义一个方法
}

      再次定义一个类

      class B {
    public void fun(A a){//定义一个方法，接收的是A类型的对象作为参数
        System.out.println("B类的fun函数"）;
        a.do();                    
    }    
}

      实现A接口的类：

      class AImpl implements A{
    public void do(){
        System.out.println("doSomeTing");
    }
}

      测试类：

      class Testing{
    public static void main(String args[]){
        B b = new B();
        //下面的是你匿名内部
        b.fun(new A(){
            public void do(){
                System.out.println("匿名内部类”）;
            }    
    });

}

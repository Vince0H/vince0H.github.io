      Java子类实例化过程和复写

      Post on 9月 18 2014

      下面的过程主要记录的是子类实例化和子类在继承父类过程中，函数的复写时，子类可以通过使用super调用。

      首先我们写出一个父类：person.java

      class person{
    String name;
    int age;
    person(){
        System.out.println("Person的无参数构造函数");
         }

    person(String name,int age){
        this.name = name;
        this.age = age;
        System.out.println("person有参数构造函数");
         }
    void study () {
        System.out.println("学习");
    }


}

      再次写出一个子类：student.java

      class student extends person{
    int grade ;
    student () {
        super();//调用父类构造函数，放在第一条语句，类似this
        System.out.println("student无参数构造函数");
    }
    student(String name,int age,int grade){
        super(name,age);//在这里，调用了父类的构造函数，放在第一句
        this.grade = grade;
    }
    void study (){
        super.study();
        System.out.println("读书");
    }
}

      说明：

      • 在子类示例化过程中，可以使用super（）；来调用父类的构造函数，并且需要放在第一句，类似this一样，复习一下，this在使用的时候可以调用同类的函数。

      • 复写（override）出现在具有父子关系的两个类当中，父类和子类各有一个函数，这个两个函数的定义（返回值了类型、函数名、参数列表）完全相同

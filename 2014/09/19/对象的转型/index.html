      对象的转型

      Post on 9月 19 2014

      这是一个非常重要的知识点，就像吃饭一样正常，下面记录的是学习的内容：

      1. 对象的向上转型
      2. 对象的向下转型

      向上转型—将子类的对象赋值给父类的引用

      student是子类，person是父类
student s = new student();
person p = s;
//上面两句可以用一句代替
person p1 = new student();

      向上转型—将父类的对象赋值给子类的引用

      //向下转型语句，先向上转型，再次向下转型
person p = new student();
student s = (student)p

      创建一个父类：person.java

      class person {
    String name;
    int age;
    void introduce(){
        System.out.println("my name is"+name+"the age is"+age);
    }
}

      创建一个子类：student.java

      class student extends person{
    String address;
    void read(){
        System.out.println("reading");
    }

    void introduce(){
        super.introduce();
        System.out.println("home-"+address);
    }
}

      创建一个测试类：test.java

      class test {
    public static void main(String args[]){

        //向上转型语句
        student s = new student();
        person p= s;
        //上面的两句可以直接用下面一举代替
        person p1 = new student();

        //向下转型语句，先向上转型，再次向下转型
        person p2 = new student();
        student s2 = (student)p2

        p.name="Zhan san";
        p.age=20;
        s.address="BeiJing";
        p.introduce();
    }
}
      • 一个引用能够调用那些成员（变量和函数），取决于这个引用的类型
      • 一个引用调用的是哪个方法，取决于这个引用所指向的对象

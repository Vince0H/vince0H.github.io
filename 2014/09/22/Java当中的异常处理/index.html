      Java当中的异常处理

      Post on 9月 22 2014

      什么是异常：中断正常指令流的事件，在运行的时候出现错误。

      异常跟语法错误是两种不同的情况，然而什么时候用哪一个异常处理方法，这需要编程经验。

      比如：try….catch….finally

      • RuntimeException + throw的作用

          class user{
      private int age;
      public static void setAge(int age){
          if(age<0){
              //自定义异常语句
              RuntimeException e = new RuntimeException("age不能为负数");
              throw e;//抛出异常，在这里终止运行
          }
      }
  }

      • try+catch+throws的作用,子类

          class user{
        private int age;
        public  void setAge(int age) throws Exception{
            if(age<0){
                //自定义异常语句
                Exception e = new Exception("age不能为负数");
                throw e;//抛出异常，在这里终止运行
            }
            this.age=age;
        }
    }

      测试类：test.java

      public class test{
    public static void main(String[] args) {
        user user=new user();
        try{
            user.setAge(-20);
        }
        catch(Exception e){
            System.out.println(e);
        }
    }
}

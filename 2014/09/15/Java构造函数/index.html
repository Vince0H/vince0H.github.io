      Java构造函数

      Post on 9月 15 2014

      在学习面向对象过程中，对于构造函数的使用非常方便，可以不用每new一个对象就要不断输入，但是个人发现，还要结合函数的重载才是最好的，请看下面的代码：

      新建一个类，dog.java

      class dog{
   dog(){
        //默认的函数，可以忽略
   }
    //下面重新构造dog（）函数
   dog(String a,int b,String c){
    name=a;
    age=b;
    color=c;
   }
   String name;
   int age;
   String color;


}

      新建一个测试类，testjava

      class test{
public static void main(String args[]){
    dog d1=new dog("wang",2,"back");//重载了构造函数，可以直接输入
    dog d2=new dog("cai",2,"white");//方便，不用一个个输入
    System.out.println(d1.name+d1.age+d2.color);//测试一下是否成功
    System.out.println(d2.name+d2.age+d2.color);

   }
}

      输出结果是：

          wang2white
    cai2white

      this的用法，结合构造函数

      首先新建一个类：person.java；在理解this的时候，我们可以把它理解为是：我，代表本身对象；
      下面有使用到构造函数。 

      class person{
    String name;
    String adress;
    int age;
    person(){
        System.out.println("没有参数的构造函数");
         }

    // the first using
    person(String name,int age){
        this();//只能放在第一行
        this.name = name;
        this.age = age;
        System.out.println("两个参数的构造函数“）；
         }
    person(String name,String adress,int age){
        this(name ,age);//继承了上面的person，只能放在第一行
        this.adress =adress;
        System.out.println("三个参数的构造函数”）；
         }

    }
